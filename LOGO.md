# Logo Placement

The website expects a logo file named `plane.png` in the root directory.

Please place your plane logo as `plane.png` in the root of this project before deploying.

The logo should be:
- PNG format
- Transparent background recommended
- Approximately 64x64 pixels or similar aspect ratio
- Will display at 32px height in the navigation

Current locations where logo is used:
- Navigation bar (index.html and all doc pages)
- Footer (index.html)
